# Holberton School — Low-level Programming

Small collection of C projects and exercises for Holberton School curriculum. Focused on fundamentals of C, memory management, data structures, and toolchain basics.

## Requirements
- Unix-like OS (Linux, macOS)
- gcc (recommended flags in examples)
- Make (optional)
- Text editor, valgrind (optional for memory checks)

## Building
Compile single files:
```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 file.c -o file
```
Use a Makefile (when present):
```bash
make
```

## Usage
Run compiled programs:
```bash
./program [args]
```
Run tests manually by executing expected binaries or using provided test scripts.

## Repository layout (example)
- 0x00-hello_world/ — first programs and compilation exercises
- 0x01-variables_if_else_while/ — control flow and variables
- 0x02-functions_nested_loops/ — functions and loops
- ... — further project directories
